(Инструкция пока неполная)

для начала в консоли cmd надо ввести
'''
pip install -r requirements.txt
'''


инструкция по первому окну - Translator:

1. По умолчанию стоят язык для перевода английский, можно нажать на эти поля и выведется окно с выбором языка
(Слева выбирается язык, с который передится, справа - на который)

2. Текст для перевода можно ввести только в левое текстовое поле (в правом можно только копировать текст)

3. Чтобы перевести текст, надо нажать на кнопку "Translate", которая почти в центре приложения

4. Имеется возможность сохранять переводы в историю поиска, если стоит галочка в квадрате возле слова "Saving"

5. Можно в левом верхнем углу нажать на "|||" и появится возможность переключиться на Историю переводов (History) и Викторину (Quiz)


инструкции по окну Истории - History:

1. В виде таблицы показывается история переводов в таком порядке: язык, с которого переводили; текст на этом языке; язык, на который переводили; полученный текст

2. Таблица разделена на страницы и слева можно выбирать страницы

3. Если история переводов мала и всего одна страница получилась, то для вывода этой страницы надо нажать на ENTER в поле ползунка


инструкция по окну Викторины - Quiz:

1. Все слова в викторине не случайны, они все взяты из истории переводов

2. Вы не можете начать, пока не выберете: из какого языка вам будет выдавать приложение и на каком языке вы будете писать. 

3. После этого вы можете нажать на кнопку "let's Start" и вам выведется случайный текст, вам нужно будет написать в поле перевод текста 

4. Когда ваш перевод будет готов, нажмите на кнопку "I've Done", которая появилась заместо кнопки "Let's Start"

5. Чтобы возобновить возможность сыграть в викторину, нажмите на кнопку "OK", которая появится позже
